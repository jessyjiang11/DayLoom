# 周计划与月计划直接添加

## 目标

用户可以在计划页的“本周计划”或“本月计划”区域直接新建计划，不必先建立待安排任务再拖入周期。

## 交互

- 周、月周期栏右侧显示“＋ 添加”按钮。
- 点击后复用统一计划编辑器。
- 周视图自动选择当前查看周，月视图自动选择当前查看月份。
- 日格中的添加入口、周期拖拽和待安排区保持原有行为。

## 数据

编辑器继续通过 `schedule_granularity`、`schedule_period_start` 和 `schedule_period_end` 保存数据，不增加新表或新字段。周计划由当前周一到周日组成；月计划由当月第一天到最后一天组成。

## 验证

- 自动化测试确认周入口传入当前周的周一。
- 自动化测试确认月入口传入当前查看日期，并由编辑器归一为月份。
- 完整运行测试、代码检查与生产构建。
