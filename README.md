# DayLoom / 不秃

> 把长期方向慢慢落实到今天。

DayLoom（中文名：不秃）是一款连接长期目标与日常执行的计划工具。目前仓库收录的是 **v0.3 Web 交互原型**，用于验证目标树、分层排期、拖拽调整和复盘体验。

## 在线体验

[打开 DayLoom / 不秃 Web 原型](https://jessyjiang11.github.io/DayLoom/)

## v0.3 已实现

- 人生方向、阶段目标、项目与普通任务组成的目标树
- 目标的增加、删除、展开、收起与自动进度
- 月、周、日三种计划视图
- 本月计划、本周计划与具体日期任务分层展示
- 任务在月份、周、日期和具体时间之间调整
- 日期格之间拖拽改期，以及拖入小时日历
- 收集箱、未规划时间和待重新安排
- 所有任务和目标共用的编辑器
- 今日重点、日复盘和用户自定义的本周一句话

## 本地体验

项目不需要安装依赖。在仓库根目录启动静态文件服务：

```powershell
python -m http.server 4173 --directory .\prototype
```

然后访问：

```text
http://127.0.0.1:4173
```

也可以直接打开 `prototype/index.html` 查看原型。

## 项目结构

```text
docs/        产品分析、PRD 与实施计划
prototype/   Web 交互原型和数据逻辑测试
```

## 运行测试

```powershell
node --test .\prototype\model.test.mjs
```

## 当前限制

- 这是用于确认产品方向的原型，不是正式生产版本。
- 页面刷新后会恢复演示数据。
- 导入、导出和清空数据目前只展示入口。
- 当前不连接服务器，也不接入 AI。
